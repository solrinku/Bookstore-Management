# 📚 Bookstore Management System

A web-based Bookstore Management System built using Django that helps manage books, inventory, customers, and orders efficiently. The system provides a user-friendly interface for administrators to maintain bookstore operations and for users to browse and purchase books seamlessly.

## Features
- User Authentication (Login/Register)
- Book Management (Add, Update, Delete)
- Search and Browse Books
- Inventory Tracking
- Customer Management
- Order Processing
- Responsive User Interface

## Tech Stack
- Python
- Django
- HTML, CSS, JavaScript
- SQLite/MySQL

## Installation

1. Clone the repository:
   ```bash
    git clone <repository-url>
   
2. Navigate to the project directory:
   ```bash
    cd bookstore-management-system
   
3. Install dependencies:
   ```bash
    pip install -r requirements.txt
   
4. Run migrations:
   ```bash
    python manage.py migrate
   
5. Start the development server:
   ```bash
    python manage.py runserver
   
6. Open your browser and visit:
   ```bash
   http://127.0.0.1:8000/
